# Pencil 原型设计 - PolarClock

## 概述
PolarClock 番茄钟时间管理系统，Web 应用，路径 `/clock/*`。

---

## 页面清单与路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 登录 | `/clock/login` | 用户名登录 |
| 首页 | `/clock/home` | 当前状态 + 快速开始 |
| 计时器 | `/clock/timer` | 全屏计时器 |
| 任务 | `/clock/tasks` | 任务列表 + 甘特图 |
| 任务详情 | `/clock/tasks/:id` | 子任务甘特图分支 |
| 黑客松 | `/clock/hackathon` | 黑客松项目管理 |
| 日程 | `/clock/schedule` | 吃饭/课程 block |
| 设置 | `/clock/settings` | 用户设置 |

### P0.5 Last Thing to Do（内嵌在任务页）

```
┌─────────────────────────────────────────────┐
│                                             │
│   "干不下去的时候，做相对最重要且最想做的"     │
│                                             │
│         想干程度 (Y轴)                       │
│              ↑                              │
│              │    ★ 当前任务               │
│              │    ● 任务A                  │
│              │         ● 任务B              │
│         0────┼────────────→                │
│              │      重要程度 (X轴)          │
│              │                              │
│                                             │
│   点击任务卡片可跳转 → 任务详情               │
│                                             │
│   ┌─────────────────────────────────────┐   │
│   │ 📋 任务A - 最优选择                   │   │
│   │    [开始番茄钟]                      │   │
│   └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

**填写流程（两次拖动）**：
1. **第一次拖动**：重要程度轴 - 所有任务按重要程度排列
2. **第二次拖动**：想干程度轴 - 所有任务按想干程度排列
3. **结果**：两个数轴的正交结果决定"Last Thing to Do"

## 扁平导航说明

所有主要页面为扁平关系，键盘左右键切换：

```
[Home] ←→ [Timer] ←→ [Tasks] ←→ [Hackathon] ←→ [Schedule] ←→ [Settings]
```

特殊跳转（Tasks 内部）：
```
[Tasks] → [Tasks/:id]（任务详情）
[Tasks] → Last Thing to Do（模态或内嵌）
```

## 组件状态

### Timer 状态
| 状态 | 显示 |
|------|------|
| `idle` | 45:00，灰色 |
| `running` | 实时倒计时，绿色 |
| `paused` | 当前时间，橙色闪烁 |
| `finished` | 00:00，红色，响铃图标 |

### 任务卡片状态
| 状态 | 样式 |
|------|------|
| `pending` | 灰色文字 |
| `in_progress` | 绿色边框 + 高亮 |
| `completed` | 划线 + 50% 透明度 |


- 归零后开始**正计时**
- 可提前结束
- 超时显示差异

## Pencil 操作步骤

1. 打开 Pencil 插件
2. 新建文档
3. 按上述线框图绘制各页面
4. 标注关键交互（点击跳转、键盘导航）
5. 在 Cursor 中反馈修改意见

---

## 设计原则

1. **功能性优先**：操作型 UI，不是展示型
2. **大字体**：计时器数字占屏幕 60%+
3. **暗色主题**：#1a1a2e 背景，#e0e0e0 文字
4. **扁平导航**：无面包屑，平等页面关系
5. **macOS 风格**：圆角、柔和阴影、拖拽手柄
