# Skill: reference/ 目录

> Distilled from: [[entity-reference-dir]]

## Prerequisites

- Understanding of reference/ 目录 fundamentals
- Familiarity with related concepts

## Core Competencies

- 适用于中大型软件项目的知识资产管理
- 适用于需要追踪技术来源和参考文献的项目
- 适用于需要进行技术对标和竞品分析的场景
- 参考资料的变更历史可完整追溯
- 团队成员可以同步最新的参考资料状态
- 可以通过分支管理不同版本的参考资料
- 将大文件的实际内容存储在 LFS 服务器上
- Git 仓库中只保存指向大文件的指针（文本引用）
- 克隆仓库时，按需下载大文件内容
- 在目录中只保存指向外部资源的 URL 链接
