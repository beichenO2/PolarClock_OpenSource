## Core Competencies

- N_total: 参考资料总数量
- N: 参考资料总数
- ⌊ ⌋: 向下取整运算
- 开发者可快速定位所需类型的参考资料
- 新成员能够系统性地了解项目技术渊源
- 大文件通过 Git LFS 或链接方式妥善管理
- 增加了资料定位的认知负担

- 维护成本显著增加
